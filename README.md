# EWU Course Schedule Tool

Welcome to the EWU Course Schedule Tool! This web application is a streamlined, user-friendly platform designed to help students at East West University elegantly view, filter, and plan their course routines and schedules.

## 🎯 Purpose of the Website
The main objective of this tool is to provide an accessible, fast, and highly responsive dashboard for checking the availability, timing, and classroom allocations of available courses. It is built to optimize the advising process and reduce friction when students are planning their semesters.

## ✨ Features
- **Modern Dashboard (Main & V2)**: Choose between our classic experience and the enhanced V2 dashboard designed with a premium, focused UI.
- **Real-Time Filtering**: Rapidly search and filter courses by subject, faculty, seat availability, timings, and class codes.
- **Dynamic Sorting**: Check available capacity for classes seamlessly.
- **Mobile Responsive**: Fully usable across smartphones, tablets, and desktops.
- **Export to PDF**: Easily export your planned courses directly into a clean, physical copy.
- **Secure Sessions**: Directly hooks into standard login mechanisms to fetch private routines.

## 📖 Basic Usage Instructions
1. Navigate to the homepage.
2. Click the **"Explore V2 version"** or standard **"Login"** prompt based on your preferred browsing experience.
3. Use your standard student credentials (Student ID format: `YYYY-S-XX-XXX`) to authenticate.
4. Once logged in, use the global search bar or dropdown filters to narrow down your desired courses.
5. Watch the dashboard update seamlessly. Select the **Available Only** toggle to only see classes with remaining seat capacity. 
6. Choose the **Export PDF** action if you wish to save your plan locally.
7. Don't forget to **Logout** when finished!
